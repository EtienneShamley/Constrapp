# Constrapp — Product Overview

## What It Is

Constrapp is an Australian-built, web-first construction project management platform for builders, contractors, and quantity surveyors. It replaces spreadsheets and disconnected tools with a single workspace covering budgets, tendering, subcontractor management, purchase orders, drawings, site photos, and reporting.

## Target Market

- Small-to-medium Australian construction companies (primary)
- Project managers, quantity surveyors, company admins
- Subcontractors and clients who need controlled read/limited access

## User Roles

| Role | Access Level |
|---|---|
| Super Admin | Platform owner — cross-company visibility |
| Company Admin | Full access within their company |
| Project Manager | Projects, budgets, contacts, drawings, POs |
| QS / Office | BOQ, budgets, forecasting, reports |
| Subcontractor | Assigned projects, drawings, POs, photos |
| Client | Dashboard, pulse view, photos, reports |

## Core Modules

- **Dashboard** — Portfolio KPIs, financial overview, active project list
- **Projects** — Create and manage projects with status, budget, team
- **Budgets** — Cost codes with committed / actual / invoiced / remaining tracking
- **BOQ & Tender Tool** — Build a Bill of Quantities, set margin/overheads, transfer to budget
- **Forecasting & Cashflow** — Income/expense curves, profit analysis per project
- **Contacts** — Subcontractors, suppliers, consultants with ABN/trade/notes
- **Purchase Orders** — Create, send, and track POs linked to cost codes
- **Drawings & Documents** — Upload, version, and mark up drawings
- **Site Photos** — Tagged photo uploads per project
- **Subcontractors** — IQ™ accountability scoring, budget vs cost code breakdown
- **Timeline** — Gantt-style schedule view with delay detection
- **Reports** — PDF/CSV exports for financial, progress, and cashflow

## Proprietary Features (Future Phases)

- **Constrapp PULSE™** — Living AI project health engine; real-time portfolio score
- **Constrapp IQ™** — AI-powered schedule, variation, and accountability intelligence
- **Constrapp Quant™** — AI quantity takeoff directly from uploaded drawing files
- **Constrapp SHIELD™** — Document integrity hashing, audit trail, and anomaly detection
